Design reasoning

We defined three different css style sheets based on device size. The default css style is for small/mobile device. If the screen size is over 768px, then mediumn screen css will be applied, if the screen size is over 992px, then the large screen style sheet will be applied. The breakpoints are choosen based on the mostly popular device sizes on the market.
As a result, the home page will render fours images in a row if screen size is over 992px, and display 2 pics in a row if screensize is over 768px. Otherwise, it will display pics vertically.

Menu System

By default, list items are displayed vertically. We use inline-block to display them horizontally (side by side). Inline-block also makes the menu responsive, If you resize the browser window, the links will automatically break when it becomes too crowded.
